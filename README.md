# 📦 Product Catalog Management System

A responsive **Product Catalog Management System** built using **HTML, CSS, and JavaScript**. This project allows users to manage products efficiently using CRUD operations while storing data in the browser with **LocalStorage**.

---

## 🚀 Features

- ➕ Add new products
- ✏️ Edit existing products
- ❌ Delete products
- 📋 Display all products dynamically
- 💾 Store product data using LocalStorage
- 🔄 Automatically load products after page refresh
- 🔍 Search products by name
- 🗂️ Filter products by category
- 💰 Filter products by price range
- ↕️ Sort products by:
  - Price (Low → High)
  - Price (High → Low)
  - Product Name (A → Z)
- 📊 Dashboard Statistics
  - Total Products
  - Total Inventory Value
  - Most Expensive Product
  - Least Expensive Product
  - Number of Products in Each Category
- ✅ Form Validation
- 📱 Responsive User Interface
